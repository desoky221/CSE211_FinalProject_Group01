# ============================================================================
# TROUBLESHOOTING
# ============================================================================

# Common Issues and Solutions:
# 1. "Cannot find module" error:
#    - Run 'npm install' in Part2_Website_Project/ directory
#    - Ensure Node.js is properly installed (check with 'node --version')
#
# 2. MongoDB connection error:
#    - Verify MongoDB is running (check with 'mongod' or MongoDB Compass)
#    - Check MONGODB_URI in .env file is correct
#    - For MongoDB Atlas, ensure IP whitelist includes your IP address
#
# 3. Port already in use:
#    - Change PORT in .env file to a different port (e.g., 3001)
#    - Or stop the process using port 3000
#
# 4. JWT authentication errors:
#    - Verify JWT_SECRET is set in .env file
#    - Clear browser cookies and try logging in again
#    - Check token expiration time in auth middleware
#
# 5. CORS errors:
#    - Ensure backend server is running
#    - Check CORS configuration in server.js
#    - Verify API URL in frontend JavaScript files matches backend URL
